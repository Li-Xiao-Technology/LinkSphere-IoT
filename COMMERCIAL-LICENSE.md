# LinkSphere-IoT 商业授权说明

## 概述

LinkSphere-IoT 采用 **双许可证模式**：

- **社区许可**：免费用于非商业场景（个人学习、技术研究、非盈利部署），详见 [LICENSE](LICENSE)
- **商业授权**：商业使用需单独取得授权，详见本文

## 授权类型

### 1. 企业内部使用授权
- 适用于企业内部生产环境部署
- 按部署实例或按用户数收费
- 包含技术支持和版本更新

### 2. OEM / 产品内嵌授权
- 适用于将 LinkSphere-IoT 嵌入到商业产品中
- 一次性授权费 + 按出货量分成（可选）
- 支持品牌定制（White-label）

### 3. SaaS / 云服务授权
- 适用于提供基于 LinkSphere-IoT 的 SaaS 服务
- 按订阅模式收费（月度/年度）
- 包含 API 接入、定制化支持

### 4. 项目交付授权
- 适用于外包/集成项目一次性交付
- 按项目一次性收费
- 包含项目期内技术支持

## 常见问题

**Q：我可以用 LinkSphere-IoT 做开源项目吗？**
A：可以。只要您的项目本身也是非商业性质的开源项目，可以在遵守社区许可的前提下使用和修改。

**Q：商业授权后我可以修改源代码吗？**
A：可以。商业授权允许您根据需要修改源代码，修改后的代码仍需遵守授权协议约定。

**Q：商业授权包含所有功能吗？**
A：是的。商业授权基于社区版公开源代码的全部功能，不区分"社区版功能"和"企业版功能"，差异主要在使用授权和服务支持。

**Q：授权是永久的吗？**
A：取决于授权类型。项目交付授权为一次性永久授权；SaaS 和企业内部使用授权通常为订阅制。具体以双方签署的协议为准。
